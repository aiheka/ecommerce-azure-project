# E-Commerce Website

## Features
- User Registration
- User Login
- Product Catalog
- Shopping Cart
- Checkout System

## Technologies Used
- Python
- Flask
- SQLite
- SQLAlchemy

## Run Locally

```bash
python app.py
```
